## 项目

### 小工具与学习项目

- **2025.04 [PubMed 搜索 MCP](https://github.com/jsfsds/pubmed_search)**  
  基于 MCP 的医学文献检索与摘要分析工具，用来快速搜索 PubMed 并整理文献摘要。

- **2026.02 [LangChain 入门小项目](https://github.com/jsfsds/LangChain-Research-Agent-Starter)**  
  面向 LangChain 1.x 初学者的研究型 Agent 示例，包含 Tools、Skills、Sub-Agent、Middleware、流式输出与 PostgreSQL 记忆。

- **2026.05 [NutUI Skill](https://github.com/jsfsds/nutui-react-taro-skill)**  
  用于辅助构建、审查和排查 NutUI React Taro 小程序页面的 Agent Skill。

### 课程项目与 Demo

- **2025.04 儿童睡眠产品 Demo（Spring Boot + Expo）**  
  软硬件结合的儿童睡眠监护灯产品 Demo，包含后端服务、移动端 App 和宣传官网。  
  [后端](https://github.com/jsfsds/dreamyou_backend) · [App 前端](https://github.com/jsfsds/dreamyou_frontend_expo) · [宣传官网](https://github.com/jsfsds/dreamyou_website)

- **2025.04-2025.08 Proguesser 电竞选手竞猜网站（已停运）**  
  基于 Django、Next.js、MySQL、Redis 的 CS / Valorant 职业选手猜谜游戏，支持单人猜测和多人实时对战。  
  [前端仓库](https://github.com/jsfsds/Proguesser-CS-Valorant-pro-guessing-game-frontend) · [后端仓库](https://github.com/jsfsds/Proguesser-CS-Valorant-pro-guessing-game-backend)

- **2025.11 [儿童外语口语学习后端](https://github.com/jsfsds/kids_language_2_backend)**  
  基于 Django、PostgreSQL、LangChain 的儿童外语口语学习后端服务。

- **2026.04 [计算车尺寸 Demo](https://github.com/jsfsds/vehicle-three-view-simulator)**  
  车辆设计课程中的三视图尺寸计算与空间模拟工具，用于验证座椅旋转、共享桌板和车身包络关系。
