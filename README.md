### Utilities and Learning Projects

- **2025.04 [PubMed Search MCP](https://github.com/jsfsds/pubmed_search)**
  An MCP-based tool for medical literature search and abstract analysis, designed to quickly search PubMed and organize paper summaries.

- **2026.02 [LangChain Starter Mini Project](https://github.com/jsfsds/LangChain-Research-Agent-Starter)**
  A research-agent example for LangChain 1.x beginners, featuring tools, skills, sub-agents, middleware, streaming output, and PostgreSQL memory.

- **2026.05 [NutUI Skill](https://github.com/jsfsds/nutui-react-taro-skill)**
  An agent skill for building, reviewing, and troubleshooting NutUI React Taro mini-program pages.

### Course Projects and Demos

- **2025.04 Children's Sleep Product Demo (Spring Boot + Expo)**
  A software-and-hardware children's sleep monitoring light product demo, including a backend service, mobile app, and marketing website.
  [Backend](https://github.com/jsfsds/dreamyou_backend) | [App Frontend](https://github.com/jsfsds/dreamyou_frontend_expo) | [Marketing Website](https://github.com/jsfsds/dreamyou_website)

- **2025.04-2025.08 Proguesser Esports Player Guessing Website (Discontinued)**
  A CS / Valorant professional player guessing game built with Django, Next.js, MySQL, and Redis, supporting single-player guessing and real-time multiplayer battles.
  [Frontend Repository](https://github.com/jsfsds/Proguesser-CS-Valorant-pro-guessing-game-frontend) | [Backend Repository](https://github.com/jsfsds/Proguesser-CS-Valorant-pro-guessing-game-backend)

- **2025.11 [Children's Foreign-Language Speaking Learning Backend](https://github.com/jsfsds/kids_language_2_backend)**
  A backend service for children's foreign-language speaking practice, built with Django, PostgreSQL, and LangChain.

- **2026.04 [Vehicle Dimension Calculation Demo](https://github.com/jsfsds/vehicle-three-view-simulator)**
  A three-view dimension calculation and spatial simulation tool for a vehicle design course, used to validate relationships among seat rotation, shared tabletop, and the vehicle body envelope.
